# 프로젝트기간
2024.11.22~2024.11.28

# 개발환경
```jdk17``` ```intellij```

# 프로젝트설명
### level1
-기본적인 입출력만 이용한 메뉴선택구현

### level2
-메뉴아이템을 객체로 관리
-생성자를통해 메뉴의 정보를 관리
-메뉴아이템객체를  리스트로 관리하고 리스트에 모든 객체를 출력 후 입력값에 따라 해당객체의 정보를 출력

### level3
-리스트를 이용한 메뉴관리
-생성자를 통해 키오스크로 메뉴리스트 전달
-키오스크는 입력에따른 흐름제어

### level4_5
-메뉴객체는 메뉴아이템을 리스트로 관리
-키오스크는 메뉴를 리스트로 관리
-메서드를 이용해서 키오스크에서 리스트로 관리할 요소인 메뉴객체를 넘김
-객체 캡슐화

### challengelevel1
-장바구니 구현
-장바구니에 추가여부확인
-장비구니에 어떤 객체가 들어있을 때,주문여부를 확인
